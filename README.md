🎮 Smash Dash

Smash Dash is a Breakout-inspired arcade game developed in Java as part of an Object-Oriented Programming course. The project focuses on applying core OOP principles such as inheritance, abstract classes, and interfaces while extending an existing codebase.

🚀 Features
Player-controlled paddle
Ball movement and collision handling
Destructible bricks
Multiple game objects with shared behavior
Object-oriented game architecture
Extensible class hierarchy
🛠️ Technologies
Java
Gradle
📚 Concepts Covered
Object-Oriented Programming (OOP)
Inheritance
Abstract Classes
Interfaces
Polymorphism
Encapsulation
Software Design Principles
